4. Explorando las posibilidades de optimizar la computación en la nube

Descubre cómo las tecnologías emergentes están redefiniendo la nube

4.1 Fog computing, edge computing y mist computing

Una mirada a las innovaciones que acercan el procesamiento al usuario

Fog computing se centra en acercar la computación a la fuente de datos; edge computing extiende este concepto a los dispositivos finales; y mist computing lleva esto aún más lejos al procesar datos a nivel de sensores.

4.2 Niveles de la nube

Utiliza estas tarjetas para crear tres donde se definan: las nubes públicas y privadas

Nube Pública

La nube pública es un modelo de computación donde los recursos, como servidores, almacenamiento y aplicaciones, son proporcionados por un proveedor externo a través de internet. Este modelo es ideal para empresas y usuarios individuales debido a su facilidad de uso, escalabilidad y ahorro de costos. Los recursos están disponibles para cualquier usuario que los necesite, y se paga únicamente por el uso que se haga de ellos, lo que permite reducir gastos iniciales de infraestructura. Además, la nube pública es altamente escalable, ya que los usuarios pueden aumentar o reducir los recursos según sus necesidades sin preocuparse por la administración o el mantenimiento, ya que estos aspectos son gestionados directamente por el proveedor. Algunos ejemplos de proveedores de nube pública son Amazon Web Services (AWS), Microsoft Azure y Google Cloud Platform. Este modelo resulta ideal para proyectos con alta variabilidad en el uso de recursos o para empresas que buscan acceso global desde cualquier dispositivo conectado.

Nube privada

La nube privada es un modelo de computación diseñado para ser utilizado exclusivamente por una organización. A diferencia de la nube pública, los recursos en la nube privada no se comparten con otros usuarios, lo que ofrece un mayor nivel de seguridad, personalización y control. Esta infraestructura puede alojarse dentro de los propios centros de datos de la empresa o ser gestionada por un proveedor externo, pero siempre estará dedicada únicamente a esa organización. La nube privada es especialmente adecuada para empresas que manejan datos sensibles, como las instituciones financieras, médicas o gubernamentales, donde la protección de la información es una prioridad. Además, permite personalizar completamente la infraestructura para adaptarse a las necesidades específicas del negocio. Aunque puede ser más costosa que la nube pública, ofrece beneficios como mayor supervisión, configuraciones a medida y un entorno más seguro para manejar información crítica o regulada.
